# gowiki
-------------
Simple wiki.

Learning to golang at [this tutorial](https://golang.org/doc/articles/wiki/).



## Usage

Access Item

http://localhost:8080/view/{item}

Edit item

http://localhost:8080/edit/{item}





## Build

```bash
go build wiki.go
./wiki
```



## Development

```bash
go run wiki.go // access -> localhost:8080
```
